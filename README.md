## Note

- DAO 模式，因为添加了 gson 作为数据操作的包，如果无法直接运行，需要添加 lib/gson/目录下的 jar 包到 Project Structure 的依赖中
- 运行游戏，可以双击`AircraftWar.jar`或者使用命令行`java -jar AircraftWar.jar`运行
